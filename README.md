# halodog-flask
Back-end Flask untuk https://github.com/daffaromero/Halodog

Di-deploy menggunakan Azure VM (Ubuntu 20.04) di URL: http://tulations.eastus.cloudapp.azure.com:5000

Route yang tersedia:

- '/' (GET): home page
- '/predict' (GET, POST): hasil prediksi foto dari URL
